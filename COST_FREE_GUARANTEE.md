## ✅ How to Run Cost-Free

```bash
# Install dependencies (free)
pip install -r requirements.txt

# Run smoke test (instant, local)
python scripts/run_benchmark.py --config configs/smoke.yaml

# Run full LoRA benchmark (hours, local)
python scripts/run_benchmark.py --config configs/default.yaml
```

**All resources are cached locally after first download** - subsequent runs have zero bandwidth cost.
